# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project builds a Machine Learning model using **Linear Regression** to predict house prices based on features like area, bedrooms, age, and location.

It demonstrates a complete ML pipeline including:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Evaluation
- Prediction

---

## 🎯 Problem Statement
Predict house prices (in ₹ Lakhs) using:
- Area (sq ft)
- Number of Bedrooms
- Age of the House
- Location

---

✔ Synthetic dataset
